# Set up virtual environment

1. Open your terminal. (You can also open a terminal via VS Code.)

2. `cd` into `dev10-classwork/week-02/exercises/04-dataclasses-apis`.

3. Create a virtual environment: `python -m venv .`.

4. Activate it.

5. Install `requests` with `pip`: `pip install requests`.

6. Deactivate.

7. In VS Code, `main.py`, select the appropriate virtual environment.
